# Number Guessing Game

A simple and interactive Python game where users must guess a randomly selected number within a specified range.

---

## ğŸ“ Description

The **Number Guessing Game** asks the user to choose a number range (between 1 and 100), then challenges them to guess a secret number within that range. After each guess, the program informs the user whether their guess is too low, too high, or correct. The game tracks the number of attempts needed to guess the number.

- Tracks number of guessing attempts

## ğŸ® How to Play

1. Run the script.
2. Input lower and upper bounds between 1 and 100 (lower < upper).
3. Guess the secret number by typing guesses.
4. Receive feedback after each guess.
5. The game ends once you guess correctly, showing your attempt count.
